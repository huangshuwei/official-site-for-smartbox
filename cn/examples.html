标题

smartbox.js 是一款基于jquery 的开源的弹出层插件，它小巧但功能强大，允许用户在一个界面同时弹出多个对话框。 弹出的内容支持多种形式，样式简洁多样。想要了解更多可以查看官方文档。查看源码以及demo可以访问我的github。

多窗口模式（同级）

支持多窗口，条件是不设置属性‘zIndex’，或者多个弹窗‘zIndex’属性设置成相同。
这是你能看到当两个窗口重叠时，点击在下面的一个， 可以让交换上下的位置。

  • 多窗口模式（同级）
  • HTML
  • JavaScript
open box1 
<div id="multipleBox_sameLevel1" style="display: none;"></div>
<div id="multipleBox_sameLevel2" style="display: none;"></div>
<a id="multipleBox_sameLevel1_open" href="javascript:void(0);">open box1</a>

多窗口模式（不同级）

此插件不仅支持多窗口同级模式，并且支持不同级的形式。要求条件是设置‘zIndex’,‘zIndex’值大的在上方，
并且要求设置遮罩层即‘isShowOverlay:true’，因为不同级模式不适用遮罩层就没有意义了。

  • JavaScript
